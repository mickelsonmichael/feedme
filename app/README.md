# feedme app

React Native + TypeScript RSS reader app that runs on Android and Web.

## Scripts

- `npm run start` - start Expo dev server
- `npm run android` - run on Android
- `npm run web` - run in browser
- `npm run test` - run Jest tests
- `npm run typecheck` - run TypeScript checks
- `npm run format:check` - run Prettier check

## Local development

```bash
npm install
npm run start
```

## Notes

- The app stores long-term data locally (SQLite where supported).
- Web-specific persistence can use local storage where appropriate.
